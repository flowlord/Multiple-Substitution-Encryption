# CDB (Character DataBase)
-------------------

Les fichier qui en pour l'extention _DB_
contient uniquement des caracères spéciaux.

ils servent de base de donées, pour générée des clés de chiffrement.
(comme un générateur de mot de passe)

_emoji.db_ contient uniquement des émojis.
_asian.db_ contient uniquement caractères issus de langue asiatique.
_lang.db_ contient uniquement caractères issus de différente langue.
_all.db_ contient tous les caractères des 3 précédents.


certain fichier peuvent faire buger votre IDE à sont ouverture.
libre à vous d'ajouter, suprimer des caractères.
